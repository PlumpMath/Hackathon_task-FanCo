About the Project

Fanco is a fictional company based in Boston that sells hand-held self-misting fans.
The purpose of this web-app is to visualize annual sales history data with correlations to seasonal weather.
Additionaly, a live weather api is utilized to provide up-to-date forecast info for sellers in the field.

This project was assigned to our team by IBM as our final Hackathon project at Elevation Academy's Coding Bootcamp.

A year of sales history was presented in the form of csv/excel files, and we were tasked with manipulating and visualizing the total sales data using the D3.js charting library. The visualizations were then to be added to a full stack SPA via AngularJS custom directives.

Additional tasks included:

  • Visualizing live weather forecast Data from an api
  • Creating visualizations to provide alternate views and insights into the data
  • Leveraging the Twilio messaging app to provide custom updates for sellers in the field

All tasks were completed succesfully in the week allotted.

The main challenge in the project was learning new technologies from scratch (D3, Twilio, custom directive integration).

By separating out tasks and working together we got it all done and had a great time too!
